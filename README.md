# InFrame Website

Marketing landing page and design documentation for InFrame.

## Preview

Once GitHub Pages is enabled for this repository, the site is available at:

**https://stephencalvillo.github.io/InFrame/**

## Project structure

- `index.html` — landing page draft
- `docs/` — design system, guidelines, Figma alignment notes

## Local preview

Open `index.html` in a browser, or run a local server:

```bash
python3 -m http.server 8080
```

Then visit http://localhost:8080
