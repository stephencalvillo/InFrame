## Two contexts in Figma

The component library targets **light-surface app UI** (white inputs, slate text, bordered controls). The landing page draft (`index.html`) uses a **dark marketing theme** (slate backgrounds, steel text, amber CTAs).

Both share core tokens (Manrope, `#EFB921`, `#222F3E`) but apply them differently. See [figma-html-alignment.md](./figma-html-alignment.md) before syncing styles blindly.
